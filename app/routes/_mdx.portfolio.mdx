### 소개

**영어 숙어를 편리하게 학습하고 더 오래 기억하기 위해 제작한 웹 어플리케이션**

<Period>2023. 03. ~ 진행 중</Period>

### 작업 사항

**프론트엔드**

- TypeScript, React 기반 프론트엔드
- React 프로젝트를 Remix로 리팩토링
  - 서버 사이드 렌더링
  - 검색 엔진 최적화
- Lighthouse Performance 점수를 50점 초반에서 70점 후반으로 개선
  - 응답 헤더에 캐시 컨트롤 설정
  - 화면에 지금 필요한 이미지만 렌더링하도록 최적화
- 스크롤 최적화하여 프론트엔드 성능 개선
  - `react-window`로 리스트, 그리드 뷰 컴포넌트에서 스크롤 가상화 작업
  - `react-window-infinite-loader`로 무한 스크롤 작업
- Web SpeechSynthesis API로 TTS 작업
  - 예문 텍스트를 클릭하면 클릭한 지점에서 음성 시작하도록 작업

**백엔드**

- Go 기반 백엔드
- 오프셋 기반 페이지네이션에서 커서 기반 페이지네이션으로 리팩토링
- OpenAI API와 연동하여 해당 숙어와 사용자가 작성한 프롬프트에 따라 예문 생성
- DB 트랜잭션으로 DB 작업 일부 실패할 때 롤백되도록 리팩토링

## Resume

### 소개

**Remix, MDX로 개발된 이력서**

<Period>2023. 04. ~ 진행 중</Period>

### 작업사항

- TypeScript, Remix 기반 프론트엔드 개발
- Cloudflare Pages로 배포
  - 깃허브 레포지터리와 연결하여 메인 브랜치 업데이트될 때마다 자동 배포
  - `@remix-run/node`에서 `@remix-run/cloudflare`로 교체
- MDX 파일 렌더링 최적화
  - 서버 런타임에서 `node:fs/promise`로 파일을 불러와 렌더링하는 방식에서 빌드 타임에서 파일을 컴파일하도록 리팩토링
  - 프로젝트 내 컴포넌트 렌더링
